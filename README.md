# Digital Twin

An AI digital twin project built with a full-stack architecture and deployed using AWS and Terraform. The repository combines backend services, frontend components, deployment scripts, and infrastructure-as-code for a production-oriented setup.

## Overview

This project explores how to build and deploy an AI-powered digital twin system using cloud infrastructure and modern web technologies. It is structured to separate application logic, UI, supporting scripts, and infrastructure provisioning.

## Tech Stack

- Python
- AWS
- Terraform
- Next.js
- TypeScript
- Shell scripting
